Code Book

This code book summarizes the resulting data fields in tidy.txt.

Identifiers

    Subject - The ID of the person being tested
    Activity - What was being done while the data was being collected.
	
Activity Labels

    WALKING : The person was walking. 
    WALKING_UPSTAIRS : The person was walking up a flight of stairs. 
    WALKING_DOWNSTAIRS : The person was walking down a flight of stairs.
    SITTING : The person was sitting.
    STANDING : The person was standing
    LAYING : The person was laying down.
